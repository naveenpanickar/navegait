# navegait EPX™ — Website Wireframe Package

## Enterprise Partnering Experience Platform

### Files
| File | Description |
|------|-------------|
| `index.html` | Landing page with vertical selector & platform overview |
| `enterprise.html` | Enterprise workspace dashboard |
| `partner.html` | Partner workspace dashboard |
| `sustain.html` | Sustain EPX — ESG & CSR vertical |
| `agri.html` | Agri EPX — Farmer & FPO vertical |
| `health.html` | Health EPX — Healthcare partnerships vertical |
| `supply.html` | Supply EPX — Supplier & manufacturer vertical |
| `tech.html` | Tech EPX — AI & technology partner vertical |
| `styles.css` | Full design system (brand colors, typography, components) |
| `components.js` | Interactions, animations & scroll effects |

### Brand System
- **Primary**: #00295A (navegait Navy)
- **Accent**: #C09020 (Gold)
- **Display font**: Fraunces (Google Fonts)
- **Body font**: DM Sans (Google Fonts)
- **Tagline**: Enterprise Partnering Experience

### To publish
Drop all files into any web host root directory. No build step required.
All assets are self-contained with embedded logo.
